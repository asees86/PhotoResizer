# Kerala PSC Photo & Signature Resizer

A free, browser-based tool to resize your **photograph** (150×200 px, ≤30 KB, with name & date) and **signature** (150×100 px, ≤30 KB) to the exact Kerala PSC Thulasi profile specifications.

Everything runs entirely in your browser — **nothing is uploaded to any server**.

## Live app

👉 **https://asees86.github.io/PhotoResizer/**

_(Available once GitHub Pages is enabled for this repo.)_

## Features

- Photo resize to 150×200 px with name & date overlay, ≤30 KB
- Signature resize to 150×100 px, ≤20 KB
- Official-style selection cropper with face guides
- Adjustable font sizes for name and date
- 300 DPI output

## Run locally

Just open `index.html` in any modern browser — no build step or server required.

## Tech

Plain HTML, CSS, and JavaScript. No dependencies.
